## Key Discoveries
- ApplicationUser has NO UserRole property — roles are ASP.NET Identity roles (AspNetRoles). Guards must load roles via _context.UserRoles.Join(_context.Roles,...) pattern.
- LetterType has NO LetterCode property — short codes stored in LetterName. Any reference to LetterCode is a bug.
- PAJAChecklist.IsComplete depends on CompletedAt.HasValue — must stamp CompletedAt before evaluating; do not evaluate IsComplete and then stamp.
- SeedDataService uses runtime seeding (SeedAsync), not migration-seeded HasData. Gap-fill block needed for existing DBs.
- Razor ASP.NET Core 10 throws RZ1010 for @{} blocks nested inside @if/@else — prefer direct C# statements in Razor code contexts.
- System.Security.Claims is NOT in implicit usings for net10.0 web SDK — must be added explicitly when using ClaimsPrincipal.FindFirstValue in controllers.
